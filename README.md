## Quick start

Prerequisites:
- Node.js (16+ recommended)
- A package manager: pnpm (preferred), npm or yarn

Install dependencies:

PowerShell
```powershell
pnpm install
# or
npm install
```

Run the development server:

```powershell
pnpm dev
# or
npm run dev
```

Build for production:

```powershell
pnpm build
pnpm start
# or
npm run build
npm start
```
